# Changelog

All notable changes to this project will be documented in this file.

## [1.0.0] - 2026-05-08

### Added
- Initial release.
- `/ie create [tag]` — Save held item to Item.yml.
- `/ie take <item>` — Get custom items from registry.
- `/ie list` — View all registered item tags.
- `/ie reload` — Hot-reload configuration without restart.
- Extended enchantment support (bypass vanilla caps).
- Custom display names, lore, damage, and unbreakability flags.
- Permission-based access control.
- Default example items in `Item.yml`.
